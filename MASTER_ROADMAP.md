
## 🔄 구현 방식

### 1. **백엔드 (signalight-engine)**
- FastAPI 엔드포인트 추가
- SQLite 스키마 확장
- 비즈니스 로직 구현

### 2. **프론트엔드 (signalight-web)**
- 새로운 페이지 및 컴포넌트
- 상태 관리 (Context/Zustand)
- Recharts 시각화

### 3. **라이브러리**
- 유틸리티 함수 (calculator.ts 패턴)
- API 클라이언트
- 타입 정의

### 4. **문서**
- 아키텍처 설명
- API 문서
- 사용자 가이드
