# Phase 11: AI & 머신러닝 (지능형 투자)

## 📋 개요
머신러닝 모델과 AI를 활용한 신호 정확도 향상 및 예측 기능

## 🎯 6개 기능 구현

### G1: AI 신호 신뢰도 점수
- 각 신호의 정확도 스코어 (0-100)
- 신호 조합 신뢰도 (RSI + MA + VWAP)
- 최근 성과 기반 가중치
- 높은 신뢰도 신호만 필터링

### G2: ML 진입/진출 예측
- 과거 데이터로 최적 진입점 학습
- 최적 진출 포인트 제시
- 모델 정확도 백테스트

### G3: 감정 분석
- 뉴스/트윗 감정 점수
- 긍정/부정 비율 그래프

### G4: 패턴 인식
- 차트 패턴 자동 감지
- 과거 유사 패턴 비교

### G5: 이상치 탐지
- 비정상 거래량 감지
- 정상범위 벗어난 지표 감지

### G6: 성격 기반 추천
- 투자자 성향 분석
- 맞춤형 신호 추천

## 🚀 구현 우선순위
G1 → G2 → G3 → G6 → (G4, G5는 선택적)
